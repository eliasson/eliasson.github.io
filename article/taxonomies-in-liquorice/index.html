Taxonomies in liquorice

Liquorice gets updated to support taxonomies in Hugo.

Posted in Hugo with tags liquorice, at Tuesday, November 11, 2014

My Hugo theme liquorice was updated yesterday to add simple support for Taxonomies. Both categories and tags is supported both as list view and as the ingress part of a single article.

In order to use it you need to configured in your site configuration (see documentation), in TOML this is basically adding:

[indexes]
    category = "categories"
    tag = "tags"

You then add the categories and / or tags per article in the front-matter like:

+++
Categories = ["Hugo"]
Description = "Taxonomies in liquorice"
Tags = ["liquorice"]
...
+++

Liquorice supports either both categories and tags, only one or neither of them.
